# Legacy Root Module

Este diretorio preserva o root module Terraform antigo e os artefactos associados.

Conteudo aqui colocado:

- deixou de ser a versao ativa do repositorio
- existe apenas para referencia historica e apoio a migracao
- pode conter sintaxe quebrada, placeholders e fluxos imperativos que a arquitetura nova removeu

A versao ativa vive agora em `inventory/`, `modules/`, `stacks/`, `env/` e `docs/`.
